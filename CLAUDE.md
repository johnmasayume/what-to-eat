# what-to-eat

A lunch place picker for the team — crowdsourced restaurant entries with daily recommendations.

## Quick start

```bash
cd backend
uvicorn main:app --reload
# visit http://localhost:8000
```

## Structure

```
what-to-eat/
├── backend/      # FastAPI + SQLite — see backend/CLAUDE.md
├── frontend/     # Vue 3 CDN single-page app — see frontend/CLAUDE.md
└── CLAUDE.md
```
